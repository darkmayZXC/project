# **App Name**: Компас Карьеры

## Core Features:

- Оценка по коду Холланда: Реализация теста Джона Холланда для самостоятельного поиска, чтобы определить тип личности пользователя на основе шести профессиональных тем (RIASEC).
- Расчет темы RIASEC: Вычисление основной темы RIASEC пользователя на основе его ответов на вопросы теста.
- Предложения по карьере: Отображение списка подходящих профессий на основе вычисленной темы RIASEC.
- Инструмент построения карьерного пути на основе генеративного ИИ: Инструмент ИИ предлагает конкретные образовательные пути, развитие навыков и анализ рынка труда для успешного продвижения по рекомендованной карьере с учетом местоположения, уровня образования и другого контекста.

## Style Guidelines:

- Основной цвет: Мягкий фиолетовый (#A098D3), чтобы вызвать чувство созерцания и руководства.
- Цвет фона: Светло-фиолетовый (#F4F2FB), слегка приглушенный.
- Акцентный цвет: Теплый розовый (#D398A0) для выделения важных элементов и призывов к действию.
- Шрифт для тела и заголовков: 'Inter' для чистого и современного пользовательского опыта.
- Используйте чистые и простые значки для представления различных областей карьеры и разделов оценки.
- Поддерживайте чистый и интуитивно понятный макет, чтобы эффективно направлять пользователей через оценку и результаты.
- Тонкие переходы и анимация для обеспечения обратной связи во время оценки и навигации по вариантам карьеры.
